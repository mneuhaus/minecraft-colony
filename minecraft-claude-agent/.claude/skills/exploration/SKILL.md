# Exploration Skill – Scouting and Resource Discovery

Scout and explore to find resources, suitable build locations, and important features.

## Available Tools

- **find_ores** - Locate ore blocks with coordinates/distances
- **find_water** - Find water sources with depth info  
- **find_flat_area** - Find flat areas for building
- **find_block** - Find any block type
- **detect_biome** - Identify current biome

## Common Tasks

**Find building site**: find_flat_area() → check water nearby → verify biome
**Find ores**: find_ores(oreType) → navigate to nearest → mine
**Scout area**: Scan for ores, water, and materials → report discoveries
